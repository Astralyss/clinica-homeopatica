Declaración de Accesibilidad

Clínica Homeopática - Fecha de última actualización: Enero 2025

Compromiso con la Accesibilidad

La Clínica Homeopática se compromete a garantizar que nuestro sitio web sea accesible para todas las personas, independientemente de sus capacidades o discapacidades. Nos esforzamos por cumplir con las pautas de accesibilidad web (WCAG) 2.1 Nivel AA.

Características de Accesibilidad Implementadas

Navegación por Teclado

Todos los elementos interactivos del sitio pueden ser navegados y activados usando solo el teclado. Los usuarios pueden usar Tab para navegar entre elementos y Enter/Espacio para activarlos.

Skip Links (Enlaces de Salto)

Se proporcionan enlaces de salto para permitir a los usuarios de lectores de pantalla ir directamente al contenido principal, saltándose la navegación.

Estructura Semántica HTML5

El sitio utiliza elementos HTML5 semánticos como <header>, <nav>, <main>, <section>, <article> y <footer> para mejorar la navegación y comprensión del contenido.

Atributos ARIA

Se implementan atributos ARIA (Accessible Rich Internet Applications) para mejorar la experiencia de los usuarios de tecnologías asistivas, incluyendo roles, estados y propiedades.

Texto Alternativo para Imágenes

Todas las imágenes incluyen texto alternativo descriptivo para usuarios con discapacidad visual.

Contraste de Colores

El sitio mantiene un contraste de colores adecuado que cumple con los estándares WCAG para facilitar la lectura de usuarios con baja visión.

Panel de Accesibilidad

Se proporciona un panel de accesibilidad que permite a los usuarios:

  • Ajustar el tamaño del texto (80% - 200%)
  • Activar modo de alto contraste
  • Reducir el movimiento de animaciones
  • Controlar el sonido

Responsive Design

El sitio es completamente responsivo y se adapta a diferentes tamaños de pantalla, facilitando su uso en dispositivos móviles y tablets.

Lectores de Pantalla

El sitio es compatible con lectores de pantalla populares como JAWS, NVDA, VoiceOver y TalkBack, proporcionando información contextual y navegación clara.

Conformidad con Estándares

WCAG 2.1 Nivel AA: Nuestro objetivo es cumplir completamente con este nivel de accesibilidad.

Sección 508: Cumplimos con los requisitos de accesibilidad federales de los Estados Unidos.

EN 301 549: Cumplimos con los estándares europeos de accesibilidad.

Limitaciones Conocidas

Algunas funcionalidades pueden tener limitaciones de accesibilidad:

  • Algunas animaciones complejas pueden no ser completamente accesibles para usuarios con sensibilidad al movimiento
  • Algunos elementos interactivos pueden requerir JavaScript para funcionar completamente

Mejoras Futuras

Nos comprometemos a continuar mejorando la accesibilidad del sitio:

  • Implementar navegación por voz
  • Agregar más opciones de personalización de colores
  • Mejorar la compatibilidad con tecnologías asistivas emergentes
